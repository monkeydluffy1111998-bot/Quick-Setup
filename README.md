**NullRoot Script Engine** is an interactive Bash setup script for Arch Linux-based systems, designed to automate installation of essential packages, AUR apps, and Flatpak setup with logging and progress feedback.

<a id="features"></a>
## Features âœ¨

- Full Installation: Pacman + AUR + Flatpak  
- Selective Installation: Pacman, AUR, or Flatpak only  
- Logging: All installation steps logged in `$HOME`  
- Status UI: Color-coded success, info, warning, error messages  
- Progress Bars: Visual feedback during installations  
- User Confirmation: Avoid accidental installs

<a id="installation"></a>
## Installation ðŸ› ï¸

1. Clone the repository:

```bash
git clone git@github.com:monkeydluffy1111998-bot/Quick-Setup.git
cd Quick-Setup
```
2. Make the script executable:

```bash
chmod +x NullRoot-Setup.sh
```
3. Run the script:

```bash
./NullRoot-Setup.sh
```
> [Tip]
The script will auto-detect your system and guide you through Pacman, AUR, and Flatpak installs.

<a id="usage"></a>
## Usage ðŸ“‹
When you run the script, you will see a menu like:

1) Full Installation (Pacman + AUR + Flatpak)
2) Only Pacman Packages
3) Only AUR Packages
4) Only Flatpak Setup
5) Exit

> Select 1-5
> Confirm before installation
> Watch progress bars and logs
